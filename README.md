# 🛒 Zepto Product Data Analysis

---

## Overview
This project is a simple SQL-based analysis of Zepto product data.  
The idea was to explore how pricing actually works across products and figure out which items offer better value.

Instead of just looking at MRP or discounts, I focused more on comparing products using **price per gram**, which gives a clearer picture.

---

## What I worked on
- Calculated price per gram to compare products fairly  
- Looked at how discounts affect actual value  
- Found products that are overpriced vs good deals  
- Cleaned the data (handled null values, checked inconsistencies)

---

## Tech Used
- PostgreSQL  
- SQL  

---

## Core Skills
- Writing analytical SQL queries  
- Using aggregations and filtering  
- Handling NULL values safely  
- Thinking in terms of business insights instead of just queries  

---

## Dataset
- Source: Kaggle  
- Contains product-level data like name, category, MRP, discount, weight, etc.
